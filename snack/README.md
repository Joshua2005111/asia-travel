# 🎲 ChinaMate - Expo Snack 预览版

## 快速测试

### 方法 1：在线导入（推荐）

1. 访问 **https://snack.expo.dev/**
2. 点击 **"Create your first Snack"** 或 **"+" 按钮**
3. 将 `App.js` 的内容复制粘贴到 Snack 的编辑器中
4. 点击底部的 **"Run"** 按钮即可预览

### 方法 2：GitHub 导入

如果你将代码推送到 GitHub：
1. 在 Expo Snack 中选择 **"Import from GitHub"**
2. 输入仓库地址
3. 选择 `snack/` 目录
